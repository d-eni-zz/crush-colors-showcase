# Crush Colors 🚗

Crush Colors is a 2D reaction game developed with Unity and C#.

## 🎮 Gameplay
- Move left and right to control the car
- Catch green balls to gain points
- Avoid red balls to avoid losing lives
- Game ends after losing 3 lives

## 🚀 Features
- Simple and addictive gameplay
- Score and life system
- Start screen and game over screen
- Restart and return to menu
- Sound effects and background music

## ▶️ How to Play
Go to the `build` folder and run:

color catch.exe
